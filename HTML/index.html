Estrutura básica do HTML

voltar

Um documento HTML começa com a tag HTML e termina com a marca /HTML.
Ele também contém um cabeçalho que descreve o título da página e um corpo
no qual se encontra o conteúdo da página.
O cabeçalho é delimitado pelas tags HEAD e /HEAD.
Já o corpo é delimitado pelas tags BODY e /BODY.

Tags em HTML

voltar

As tags são usadas para informar ao navegador a estrutura do site.
Elas serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página.
As tags HTML são divididas em dois tipos: as que precisam de fechamento e as que não precisam de fechamento.

Atributos Básicos

voltar

Atributos são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento.
Os atributos HTML são um modificador de estrutura e funcionalidade de um elemento HTML.

Textos

voltar

Para escrever textos em HTML temos as tags de título conhecidas como h1, h2, h3, h4, h5 e h6.
Elas servem para marcar textos como títulos, subtítulos e assim por diante seguindo uma hierarquia e possuem tamanhos diferentes.
Outras tags importantes são as p que identificam os parágrafos do texto.

Listas em HTML

voltar

As listas permitem criar tópicos de textos para uma melhor exemplificação de um determinado assunto.
São bem úteis na criação de menus num site por exemplo. As listas podem ser ordenadas, não ordenadas e de definição.
